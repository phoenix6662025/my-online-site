Как разместить сайт на Vercel

1

Создайте Git репозиторий

Инициализируйте Git в папке проекта и загрузите код на GitHub

git init && git add . && git commit -m "Initial commit"
2

Зарегистрируйтесь на Vercel

Используйте ваш GitHub аккаунт для быстрой регистрации

Перейдите на vercel.com и нажмите "Sign Up"
3

Импортируйте проект

Выберите ваш репозиторий из списка и нажмите "Deploy"

Проект автоматически соберется и будет размещен
4

Получите ссылку

После деплоя вы получите уникальную ссылку на ваш сайт

https://my-online-site.vercel.app
